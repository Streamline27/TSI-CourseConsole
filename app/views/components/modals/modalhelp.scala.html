Help

1)To perform add delete and update operation select appropriate page and push corresponding button.

2)Use combo box on lessons and groups pages to perform search with criteria.

3)Use browser print button to create report.

4)If some field is empty when performing update or create operations, you will get warning message and operation wont take place.

5)If you try to create entity with duplicate primary key, you will get warning and operation wont take place.

6)If you delete or update discipline all the corresponding to discipline courses will be also deleted or updated.

7)If you delete or update course all the corresponding lessons will be also deleted or updated.
